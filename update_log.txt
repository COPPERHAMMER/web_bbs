# 本论坛使用 Flask + jinja + Redis + Mysql 开发，使用 Nginx + gunicorn(+gevent) 部署于腾讯云


# 优化/功能更新 2019.07.02
**1.性能优化：大部分查询操作实现中间件redis缓存，尽可能实现数据重用，减少数据库查询次数**
- 经初步测试2000次数据请求，只需进行不到100次数据库检索。

**2.安全：csfr防御/xss防御**
- 为发回复，删回复，发主题，删主题，改设置，发私信等敏感操作增加csrf防御。
- csrf token 存放于redis，以便于使用多进程负载均衡。

**3.系统通知功能，被@，主题被其他人回复时将收到通知。**
- 通知分为已读和未读，可一键标记未读通知为已读，一键清理已读通知；
- 导航栏显示未读通知条数。

**4.支持 @用户功能。**
- 中被@用户的显示效果为超链接，可访问主页；
- 被@的用户收到系统通知。

**5.支持话题和回复的删除，话题编辑功能。**

**6.增加私信功能。**
- 导航栏显示未读私信条数；
- 私信支持Markdowm 语法；；
- 私信详情页面增加回复私信入口；
- 支持收件箱，发件箱，一键设为已读功能

**7.优化话题显示**
- 用户主页最近创建的话题、最近参与的话题显示最近5个，其余可查看更多访问。
- 话题按照最后回复时间倒序显示，无回复则以创建时间为准
- 回复层增加回复用户头像显示

**8.注册时要求提供邮箱，需要验证邮箱验证码**

**9.忘记密码。重设密码：需提供注册邮箱发送验证邮件**
